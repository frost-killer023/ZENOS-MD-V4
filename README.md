# 🤖 ZENOS-MD-V1

> Bot WhatsApp multiservices privé · 247 commandes · Node.js + Baileys

## ⚡ Démarrage rapide

```bash
npm install --legacy-peer-deps
node index.js
```

Scanne le QR code ou entre le **pairing code** affiché dans WhatsApp > Appareils connectés > Coupler avec numéro.
